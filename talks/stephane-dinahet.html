Construire une API en Java Spring Boot pour son portfolio : retour d'expérience avec Loto Tracker

Dans cette présentation, je partagerai le retour d'expérience de Loto Tracker, une application web full-stack permettant de suivre les résultats du Loto et de vérifier automatiquement les tickets. Le projet combine un backend Java Spring Boot, un frontend web interactif, et une API REST sécurisée avec JWT, utilisant PostgreSQL et MongoDB pour la gestion des données. L'objectif est de montrer comment transformer un projet personnel en application complète et valorisable dans un portfolio développeur.

Notes

Technologies: Java 21, Spring Boot, Spring Security, JWT, PostgreSQL, MongoDB. Déploiement: serveur Linux avec Apache2, reverse proxy, SSL et Cloudflare. Site: https://loto-tracker.fr
