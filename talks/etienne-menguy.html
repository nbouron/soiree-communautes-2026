Etienne Menguy
Cloud Operations Architect — Amazon Web Services (AWS)
AWS Rennes

AWS DevOps Agent : votre coéquipier IA pour résoudre les incidents en minutes, pas en heures

AWS DevOps Agent (GA le 31 mars 2026) est un agent IA autonome qui apprend la topologie de vos applications, corrèle télémétrie, code et déploiements, et déclenche automatiquement une investigation à chaque incident.

Nous verrons comment il réduit le MTTR, s'intègre à vos outils (CloudWatch, Datadog, GitHub, Slack, ServiceNow…) et passe de la réaction à la prévention grâce à l'analyse des incidents passés — avec un focus sur les nouveautés GA.
